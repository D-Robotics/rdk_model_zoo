## 分割

### 概念说明
- **语义分割（Semantic Segmentation）**：区分出不同类别，用 mask 进行像素级标记（不区分同类别的不同实例）。
- **实例分割（Instance Segmentation）**：结合目标检测和语义分割的更高层级任务，区分出不同实例，用 mask 进行标记（既分类又分个体）。
- **目标检测（Object Detection）**：区分出不同实例，用 bounding box 进行目标定位。
