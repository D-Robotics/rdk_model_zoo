# UNet-resnet50 模型评估

[English](./README.md) | [简体中文](./README_cn.md)

## 概述

本目录用于存放 UNet-resnet50 语义分割模型的评估工具与脚本。

## 评估指标

典型的语义分割评估指标包括：

- **像素准确率（PA）**：正确分类像素所占的比例
- **平均像素准确率（mPA）**：逐类别的像素准确率平均值
- **平均交并比（mIoU）**：分割质量的主要评价指标
- **频权交并比（FWIoU）**：按类别频率加权的 IoU

## 使用方法

评估脚本和详细教程将在此提供。目前，用户可以使用以下标准语义分割评估库对模型进行评估：

- `mmsegmentation`
- `torchmetrics`
- 基于 VOC 或 Cityscapes 协议的自定义评估脚本

## 注意事项

- 本模块为可选模块。
- 训练和评估的最佳实践请参考原始 UNet 仓库。
