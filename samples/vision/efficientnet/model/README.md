# 模型文件

本目录包含 EfficientNet-Lite 系列模型下载脚本。

## 运行方式
```bash
bash download_model.sh
```

下载成功后，当前目录下将生成以下文件：
- `efficientnet_lite0_224x224_nv12.hbm`: EfficientNet-Lite0 模型 (224x224)
- `efficientnet_lite1_240x240_nv12.hbm`: EfficientNet-Lite1 模型 (240x240)
- `efficientnet_lite2_260x260_nv12.hbm`: EfficientNet-Lite2 模型 (260x260)
- `efficientnet_lite3_300x300_nv12.hbm`: EfficientNet-Lite3 模型 (300x300)
- `efficientnet_lite4_380x380_nv12.hbm`: EfficientNet-Lite4 模型 (380x380)
