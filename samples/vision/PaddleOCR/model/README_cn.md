# 模型管理说明

本目录用于存放已编译完成、可直接在 RDK 平台上部署的 BPU 静态模型（`.hbm` 或 `.bin` 文件）。

- **下载模型**：
  直接运行 `download_model.sh` 脚本。
  ```bash
  ./download_model.sh
  ```
