# 模型下载方式

> ⚠️ **平台说明**：LaneNet 模型仅支持 **RDK S100** 平台，不支持 RDK S600。

- 直接运行 `download_model.sh` 即可将转换好的 hbm 模型下载到此目录：

    ```bash
    ./download_model.sh
    ```

- 模型文件：`lanenet256x512.hbm`（S100 平台专用）
- 下载来源：`https://archive.d-robotics.cc/downloads/rdk_model_zoo/rdk_s100/Lanenet/lanenet256x512.hbm`
